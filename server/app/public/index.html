模块概览

日常随笔记录了博主在平时生活中浏览掘金、知乎、MDN等学习网站时候发现的一些好文章，博主打算分为以下几个模块：前端基础、框架解读、实战技巧、Webpack、Git、性能优化、计算机基础。

前端基础

既然选择了前端这个领域，那么JavaScript,CSS,HTML三剑客就必不可少，坚实的理论基础使我们能保持高效率的开发，避免或减少加班的时间(*￣3￣)╭。这一模块将会记录一些前端领域许多经典的题目或是Bug，会从简单到详细，从浅到深去理解一些重要的知识点。

框架解读

当下有许多优秀的前端框架，解读框架原理有助于帮助我们提升实战技巧，借鉴学习作者的思想。

实战技巧

实战技巧主要涉及到一些工作中常见组件库的实现过程，我们也需要有一定造轮子的能力。

Webpack

Webpack是前端项目的打包利器，这部分我们将会从打包原理、插件loader编写和实战搭建三个部分进行探索。

Git

Git是所有开发工程师都需要学会的技能（SVN瑟瑟发抖）。

性能优化

性能优化极其重要，这部分我们将会借助工具进行性能优化。

计算机基础

计算机基础是前端面试近两年的考察重点，也其中算法涉及的最多，所以这部分的重点在于算法。

另一个重点是网络部分，HTTP协议也是必不可少的。

这部分会分为这几个部分： 数据结构、算法、网络。
